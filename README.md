# TP5: Application GPS
TP réalisé majoritairement en classe, et à finir chez soit. Le but est de concevoir, en suivant les questions, un prototype de GPS optimisé pour différents profils d'utilisateurs.

## Exécution

Le fichier s'exécute via ./Exercice

## Compilation

La compilation est réalisée à l'aide d'un makefile, utilisant le compilateur g++ pour C++20.

## Tests

Le fichier main.cpp implémente tous les tests normalement, et les sorts dans la console, pas besoin de modifier quoique ce soit.

Le BFS a été réalisé à l'aide des deques de la librairie standard. Une implémentation naïve de l'algorithme a été réalisée

## Réalisation des questions

Les questions 7 et 8 n'ont pas été réalisée par manque de temps, mais celles-ci étaient optionnelles, donc l'enjeu est moindre.
